Open Trivia Database Questions

  • Click any answer to check your guess.
  • Click "New Question" to get a new question.
  • Click on the category label so select the categories (selecting none will be handle as if all were selected).
  • Click on the difficulty label so select the difficulty.
  • Check 'Force unique questions' to let the API remember all the questions you already answered. The set of previous questions is identified via a token. Your browser will remember this token even if you restart it. To reset the pool of questions uncheck this setting or wait 6h after your last request (by that time the data is not available on the server anymore).
Visit the GitHub Repo for more information.
